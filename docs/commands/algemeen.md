# Algemene commands

Hier staan de commands die iedereen kan gebuiken.
voor alle commands moet er een `/` voorbeeld: `/binds`.

# binds

```lua
/binds
```

Hier kan je al je keybinds instllen

# Animatie menu

```lua
/am
```

# Bekijk baan

```lua
/baan
```

# Report

```lua
/report
```

# OOC

```lua
/ooc
```
